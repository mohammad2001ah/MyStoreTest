# 🛒 MyStoreTest

A simple full-stack e-commerce testing project built for practicing **manual testing, Selenium automation, and API testing (Postman)**.  
This project helps demonstrate real-world QA skills including UI testing, test case design, and API validation.

---

## 📌 Project Overview

This project simulates a basic e-commerce system where users can:

- Register and login
- Browse products
- Add items to cart
- Checkout process
- Perform user authentication

The goal of this repo is to practice:
- Manual test case writing
- Selenium automation testing (Java)
- API testing using Postman
- Pre-request and post-response scripting

---

## 🧪 Testing Types Covered

### 1. Manual Testing
- Functional test cases
- Negative scenarios
- UI validation
- Authentication testing

### 2. Automation Testing (Selenium)
- Login automation
- UI interaction tests
- Element validation
- Assertions for verification

### 3. API Testing (Postman)
- Login API testing
- Create user API testing
- Pre-request scripts
- Post-response validation
- Status code verification
- Response body validation

---

## ⚙️ Tech Stack

- Java (Selenium WebDriver)
- TestNG
- Maven (if used)
- Postman

- ChromeDriver
